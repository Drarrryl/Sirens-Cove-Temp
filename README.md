# SirensCove

Developed with Unreal Engine 5

Making changes to the project:

- Go into console and use the following commands:
  - git pull origin master
  - git branch [Name of the branch you want to create] (example: git branch DarrylBranch)
  - git checkout [Name of the branch you created]
- Then edit the project

Uploading your changes to the GitHub repository:

- Click Revision Control in the bottom right and click "Submit changes"
- Then go into your console and use command git push origin [Name of the branch you created]
